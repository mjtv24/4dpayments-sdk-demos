# 4D Shipping SDK 2024 .NET Edition

This folder contains the local installation assets for the 4D Shipping SDK 2024 .NET Edition.

## Included content

- `help/DPayments.DShippingSDK.htm` — the local reference help page for the Shipping SDK.
- `help/html/` and `help/lib/` — supporting assets required by the local help system.
- `readme.txt` — the original Shipping SDK installation readme.

## Documentation reference

Use the 4D Shipping SDK online code documentation as the primary reference for integration:

- https://cdn.4dpayments.com/help/DXJ/cs/

The local help file is copied from the installed SDK and can be used offline for the Shipping SDK reference.
