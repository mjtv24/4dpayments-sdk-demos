# E-Payment Integrator 2024 .NET Edition

This directory contains the copied help and library files from the local E-Payment Integrator 2024 .NET Edition installation.

## Contents

- `help/` — HTML reference documentation for the .NET edition
- `lib/` — runtime libraries for the E-Payment Integrator .NET Edition
- `readme.txt` — original installation readme file

## Integration Notes

- The files in this folder are copied from the local installation at:
  `C:\Users\admin\OneDrive\Pictures\Documents\E-Payment Integrator 2024 .NET Edition`
- This is intended to supplement the existing 4D E-Banking SDK demo workspace with the E-Payment Integrator reference docs and runtime libraries.
- If you want, you can add project references or sample code that targets the E-Payment Integrator libraries from this folder.
